# Fire smoke detection on Raspberry Pi4
A detection project using YOLO and edge impulse. 
On-device deployment and inference are presented.

The D-Fire dataset can be found:
https://github.com/gaia-solutions-on-demand/DFireDataset

Some functions in utils.py are from:
https://github.com/gaia-solutions-on-demand/DFireDataset/blob/master/utils/utils.py

The yolo_detect.py is from:
https://github.com/EdjeElectronics/Train-and-Deploy-YOLO-Models/blob/main/yolo_detect.py

The video clip for inference is from:
https://www.youtube.com/watch?v=nBZs2_hezdY
